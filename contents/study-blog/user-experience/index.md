# 사용자 경험(UX, User Experience)

사용자가 시스템/서비스를 이용하면서 느끼고 생각하게 되는 총제적인 경험

만족, 참여, 사용, 관찰, 상호 교감을 통해 알 수 있는 가치 있는 경험

- 특징
  - 주관성(Subjectivity)
  - 정황성(Contextuality)
  - 총체성(Holistic)
