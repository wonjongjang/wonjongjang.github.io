# 프레임워크(Framework)

특정 기능을 수행하기 위해 필요한 클래스/인터페이스 등을 모아둔 집합체

|    특징     |                   설명                   |
| :---------: | :--------------------------------------: |
|  Structure  | 개발자가 코드를 작성할 수 있는 구조 제공 |
| Opinionated |                 자유도↓                  |

# 라이브러리(Library)

효율적인 개발을 위해 자주 사용하는 함수/데이터들을 미리 만들어 모아둔 집합체

|          특징          |        설명        |
| :--------------------: | :----------------: |
|     Re-usable code     | 재사용 가능한 코드 |
| Specific functionality |     특정 기능      |
|      Time saving       |     시간 절약      |
|     Unopinionated      |      자유도↑       |
